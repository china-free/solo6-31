## 1. 产品概述

世界地理国旗配对游戏是一款寓教于乐的互动学习应用，通过拖拽配对的方式帮助用户学习世界各国国旗知识。

- 目标用户：对世界地理感兴趣的学生、地理爱好者、休闲游戏玩家
- 产品价值：通过趣味游戏化方式，让用户在轻松愉快的氛围中学习50个国家的国旗，提升地理知识储备

## 2. 核心功能

### 2.2 功能模块

1. **游戏主界面**：国家名称列表、国旗卡片区域、计时器、进度指示
2. **拖拽配对系统**：国旗拖拽、配对检测、正确/错误视觉反馈
3. **结果统计面板**：完成时间、正确率、重新开始按钮
4. **题库系统**：50国数据、随机抽取算法、题目打乱

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 游戏主界面 | 国家名称列表 | 左侧显示5个随机国家名称，作为配对目标区域 |
| 游戏主界面 | 国旗卡片区域 | 右侧显示5个打乱顺序的国旗卡片，支持拖拽操作 |
| 游戏主界面 | 顶部状态栏 | 显示游戏标题、计时器、当前进度 |
| 游戏主界面 | 结果弹窗 | 显示完成时间、正确率、重新开始按钮 |

## 3. 核心流程

用户打开游戏 → 系统随机抽取5个国家并打乱国旗顺序 → 用户拖拽国旗到对应国家名称 → 系统判断配对结果（正确变绿/错误变红）→ 全部配对完成 → 显示用时和正确率 → 用户可重新开始游戏

```mermaid
flowchart TD

    F --> H{"全部配对完成？"}
    G --> D
    H -->|是| I["显示完成时间和正确率"]
    I --> J{"用户重新开始？"}
    J -->|是| B
    J -->|否| K["游戏结束"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝色 (#1e3a5f) 作为背景，搭配金色 (#fbbf24) 作为点缀色，营造世界地图的探索感
- **辅助色**：成功绿 (#10b981) 表示正确，警告红 (#ef4444) 表示错误
- **按钮风格**：圆角卡片式按钮，带悬停微缩放效果，阴影层次
- **字体**：使用 Playfair Display 作为标题字体（优雅衬线），Lato 作为正文字体（清晰易读）
- **布局风格**：左右分栏卡片布局，卡片式容器，深度阴影，圆角设计
- **装饰元素**：地球纹理背景，微妙的世界地图图案，国旗悬停发光效果

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|-------|
| 游戏主界面 | 顶部状态栏 | 深色背景卡片、渐变色标题、数字计时器、进度条 |
| 游戏主界面 | 国家名称列表 | 5个卡片式目标区域，悬停高亮，配对后变色锁定 |
| 游戏主界面 | 国旗卡片区域 | 国旗图片卡片，拖拽阴影，悬停放大，拖拽时半透明 |
| 游戏主界面 | 结果弹窗 | 毛玻璃背景，动画进入效果，统计数据大字展示 |

### 4.3 响应式设计

- 桌面端（默认）：左右分栏布局，左侧国家列表，右侧国旗区域
- 平板端：保持左右布局，适当缩小间距
- 移动端：上下堆叠布局，上方国家列表，下方国旗区域，支持触摸拖拽
